This project teaches a computer to predict salaries by looking at real data. It uses a method called "gradient descent" to find the best line that fits through the salary data.

In simple terms:
- You give it data about people's experience and salary
- It learns the relationship between them
- It draws a line showing the pattern
- You can use that line to predict new salaries

## What You Need

- Python 3
- pandas (for reading data)
- matplotlib (for drawing graphs)
